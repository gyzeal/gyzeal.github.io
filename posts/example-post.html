关于这个博客

这个博客使用纯静态HTML构建，简单而优雅。主要分享以下几个方面的内容：

  • 艺术类 - 艺术创作与欣赏，美学思考
  • Agent开发类 - AI Agent 开发实践与探索
  • RL学习类 - 强化学习理论与应用

如何使用

这个博客系统非常简单易用：

  1. 复制 posts/article-template.html 创建新文章
  2. 编辑文章内容
  3. 在相应的分类页面添加文章链接
  4. 在首页添加到"最新文章"列表

代码示例

博客支持代码高亮显示。例如一段Python代码：

def hello_world():
    print("Hello, World!")
    return True
