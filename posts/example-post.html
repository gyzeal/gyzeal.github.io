欢迎来到我的博客

Date: November 6, 2025 Category: 艺术类

欢迎来到我的个人博客！这是一篇示例文章，展示博客的基本格式和样式。

关于这个博客

这个博客使用纯静态HTML构建，简单而优雅。主要分享以下几个方面的内容：

  • 艺术类 - 艺术创作与欣赏，美学思考
  • Agent开发类 - AI Agent 开发实践与探索
  • RL学习类 - 强化学习理论与应用

如何使用

这个博客系统非常简单易用：

  1. 复制 posts/article-template.html 创建新文章
  2. 编辑文章内容
  3. 在相应的分类页面添加文章链接
  4. 在首页添加到文章归档列表

代码示例

博客支持代码显示。例如一段Python代码：

def hello_world():
    print("Hello, World!")
    return True

总结

这个博客系统简单但功能完整，适合快速开始写作。如果未来需要更多功能，可以考虑迁移到Jekyll等静态网站生成器。
