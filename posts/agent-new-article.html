1.1 权重加载到 GPU 显存

首先将权重（.pth、.safetensors）加载进 GPU 显存（VRAM）中。 模型权重的数据类型（fp32、bf16、fp16、int8 等） 决定了占用多少显存。

显存估算公式：

对于一个具有 \(n\) Billion 参数的模型，其权重占用显存约为： 


Param Memory ≈ n × 10^9 × Bytes(fp32 / bf16 / fp16 / int8 ...)

dL/dW = dL/dY · X

Activation Memory ≈
